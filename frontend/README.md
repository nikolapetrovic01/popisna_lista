# Frontend

Frontend dio projekta razvijen je korištenjem **Angular** frameworka.

## Pokretanje aplikacije u razvojnom okruženju

Za pokretanje frontend aplikacije u razvojnom režimu koristi se sljedeća komanda:

```bash
ng serve
```

Nakon pokretanja, aplikacija je dostupna na adresi:

```text
http://localhost:4200/
```

Aplikacija će se automatski ponovo učitati nakon svake izmjene u izvornom kodu.

## Build aplikacije

Za kreiranje produkcijskog build-a koristi se:

```bash
ng build
```

Generisani fajlovi bit će sačuvani u direktoriju `dist/`.
